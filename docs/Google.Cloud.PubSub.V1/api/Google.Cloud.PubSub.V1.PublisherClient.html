Create(TopicName, IEnumerable<PublisherServiceApiClient>, PublisherClient.Settings)

Create a PublisherClient instance associated with the specified TopicName. The gRPC Channels underlying the provided PublisherServiceApiClients must have their maximum send and maximum receive sizes set to unlimited, otherwise performance will be severly affected, possibly causing a deadlock. The default settings are suitable for machines with high network bandwidth (e.g. Google Compute Engine instances). If running with more limited network bandwidth, some settings may need changing.

ShutdownAsync(CancellationToken)

Shutdown this PublisherClient. Cancelling hardStopToken aborts the clean shutdown process, and may leave some locally queued messages unsent. The returned System.Threading.Tasks.Task completes when all queued messages have been published. The returned System.Threading.Tasks.Task cancels if hardStopToken is cancelled.
